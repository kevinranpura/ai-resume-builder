# ResumeWiz - AI Powered Resume Builder

AI Resume Builder is a web-based application that allows users to effortlessly create professional resumes using customizable templates and themes. Powered by AI, it helps users auto-generate resume content based on job roles and experience, streamlining the resume writing process. The platform supports real-time editing, multiple resume management, and secure cloud storage, offering a smooth and personalized user experience for job seekers.

## Tech Stack

- **Frontend**: React, TailwindCSS, React Router, Context API
- **Backend**: Firebase (Auth), Express, MongoDB
- **AI**: Groq API (LLaMA model)
- **Deployment**: Vercel (Frontend), Render (Backend)


## Installation

1. Clone the repository

```bash
  git clone https://github.com/kevin-ranpura/ai-resume-builder.git
  cd ai-resume-builder
```
2. Install Dependencies

For Frontend:

```bash
  cd frontend
  npm install
```
For Backend:

```bash
  cd backend
  npm install
```  
3. Set Up Environment Variables

4. Run the Servers:
```bash
  npm run dev
  node server.js
```
